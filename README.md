# 📖 About This Repository

This repository documents my complete hands-on learning journey while exploring Google's modern AI Agent ecosystem.

Throughout this learning journey, I built practical AI agents using Google ADK, integrated Gemini models, implemented Human-in-the-Loop approval workflows, secured agents against prompt injection, developed ambient event-driven agents, and learned how production-ready AI agents are structured.

This repository serves as both my learning notebook and portfolio.

- PII redaction
- Prompt injection detection
- Ambient event support
